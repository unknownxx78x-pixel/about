Portfolio pribadi Habibi — dibangun dengan tema **LynX GUI Library** (dark, orange–purple).

## Hosting di GitHub Pages

1. Buka **Settings → Pages** di repo ini.
2. Pada **Source**, pilih branch `main` dan folder `/ (root)`, lalu **Save**.
3. Situs akan tersedia di: `https://unknownxx78x-pixel.github.io/about/`
